# マンマシンチャート簡単作成アプリ

## 概要

本アプリは、製造現場における人と機械の時間関係を簡単に記録し、マンマシンチャートとして可視化するためのStreamlitアプリです。

対象は、自動で稼働する機械を中心とした工程です。

作業者の状態と機械の状態を記録し、Excel形式でログ、サマリー、チャートを出力できます。

---

## 主な機能

- 標準モードによる簡易記録
- 詳細モードによる作業分類記録
- 機械停止中の作業ボタン設定
- 機械稼働中の作業ボタン設定
- 監視・待ち時間の記録
- 不要なボタンの非表示
- 作業名の直接追加
- Excel出力
- CSV出力
- summaryシートによる自動集計
- logシート上のマンマシンチャート表示

---

## 状態区分

本アプリでは、以下の3状態を中心に可視化します。

| 色 | 状態 | 意味 |
|---|---|---|
| 青 | 機械停止中の作業 | 人が作業しており、機械は停止している |
| 緑 | 機械稼働中の作業 | 人が作業しており、機械も稼働している |
| グレー | 監視・待ち | 人は監視または待機しており、機械は稼働している |

---

## 想定用途

- 製造現場のIE分析
- マンマシンチャート作成
- バリ取り・検査・段取り作業の分析
- 人員配置の見直し
- 機械停止時間・監視時間の把握
- 中小製造業の現場改善支援
